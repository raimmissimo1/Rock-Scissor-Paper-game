This is a simple console-based game of Rock, Paper, Scissors written in Python.
The program allows the user to play against the computer in an interactive match.
The goal is to reach 3 wins before the computer does.

The player chooses between Rock, Paper, or Scissor.

The computer randomly selects its move using the random module.

The program compares both choices and determines the winner of the round.

Rock beats Scissor

Scissor beats Paper

Paper beats Rock

Scores are updated after each round and displayed on the screen.

The first to reach 3 wins is declared the overall winner.

The user can quit the game anytime by typing "q".
